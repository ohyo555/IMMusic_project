#### 주요기능
<hr>

- 소개
  * 기업 소개
- 연주자
  * 연주자 소개
- 뉴스 & 연주
  * 연주 정보
  * 최근 뉴스
    + 기업 블로그 실시간 크롤링
- 문의
  * 연주자 가입신청
    + 프로필, 이력 파일 업로드
    + 완료 후 문자 전송 
  * 연주문의
    + 완료 후 문자 전송 
  * 개발자 contact

#### API
<hr>

- Coolsms: 결제 후 완료 인증 문자 전송을 위한 API 
- Daum postcode: 회원가입, 멤버쉽 가입 등 주소를 받기 위한 API
